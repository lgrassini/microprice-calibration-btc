# MicroPrice calibration for BTCUSDT

This repository explores the calibration of the MicroPrice (Stoikov, 2017) high-frequency estimator
for use with crypto trading pairs such as BTCUSDT.
For more information about MicroPrice, please refer to Sasha Stoikov’s paper:

[https://papers.ssrn.com/sol3/papers.cfm?abstract_id=2970694
](https://papers.ssrn.com/sol3/papers.cfm?abstract_id=2970694)
